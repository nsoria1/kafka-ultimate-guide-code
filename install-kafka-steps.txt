Steps to run
1. Build the docker image: docker build -t ubuntu-kafka .
2. Run a container based on the image: docker run -i -t ubuntu-kafka /bin/bash
3. Set the JAVA HOME variable: export JAVA_HOME="$(update-alternatives --query java | grep 'Value: ' | grep -o '/.*/jre')"
3. Start the Zookeeper installation: /usr/local/zookeeper/bin/zkServer.sh start
4. Test the Zookeeper installation: telnet localhost 2181 from inside the container
5. Run Kafka server: usr/local/kafka/bin/kafka-server-start.sh -daemon /usr/local/kafka/config/server.properties
6. Test Kafka by creating a topic: /usr/local/kafka/bin/kafka-topics.sh --create --zookeeper localhost:2181 --replication-factor 1 --partitions 1 --topic test
7. Test Kafka by validating a topic: /usr/local/kafka/bin/kafka-topics.sh --zookeeper localhost:2181 --describe --topic test
8. Test Kafka by producing a message to a test topic: /usr/local/kafka/bin/kafka-console-producer.sh --broker-list localhost:9092 --topic test
9. Test Kafka by consuming a message from a test topic: /usr/local/kafka/bin/kafka-console-consumer.sh --zookeeper localhost:2181 --topic test --from-beginning
